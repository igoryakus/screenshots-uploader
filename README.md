# screenshots-uploader

**Screenshots-Uploader** — это легкая утилита, которая автоматически загружает ваши скриншоты на личный VPS через SSH/SFTP и мгновенно копирует ссылку для просмотра в буфер обмена. Сохраняйте полный контроль над своими медиафайлами, не полагаясь на сторонние фотохостинги.

### Возможности
* **Мгновенная загрузка:** Отправляйте скриншоты на сервер в тот же момент, когда файлы записываются на диск.
* **Автокопирование URL:** Готовая к отправке ссылка генерируется сразу и вы можете ей пользоваться, пока в фоне файл копируется на сервер.
* **Приватность на своем сервере:** Ваши изображения остаются в безопасности на вашем собственном VPS под вашим полным контролем.
* **Максимальная скорость:** Мгновенная прямая загрузка на VPS без задержек публичных хостингов.
* **Отсутствие рекламы:** Полное отсутствие баннеров, всплывающих окон, вотермарок и капч.
* **Вечное хранение:** Ваши файлы размещены навсегда и никогда не удалятся автоматически.
